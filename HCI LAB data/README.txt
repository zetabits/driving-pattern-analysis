HCILAB DRIVING DATA SET
-----------------------
Authors: Frederik Heinrich, Stefan Schneegass, Bastian Pfleging, Nora Broy,
Albrecht Schmidt

Data set website: http://www.hcilab.org/autmotive


IMPORTANT NOTE
--------------
If you use this dataset, you agree to the license information as provided in the
license file (LICENSE.txt and accompanying files).

If you would like to use the data set but need a different license for your
specific use case, please contact us!

For details about the dataset, please have a look at our website
http://www.hcilab.org/automotive and our corresponding publication:

Stefan Schneegass, Bastian Pfleging, Nora Broy, Albrecht Schmidt, and Frederik
Heinrich. 2013. A data set of real world driving to assess driver workload. In
Proceedings of the 5th International Conference on Automotive User Interfaces
and Interactive Vehicular Applications (AutomotiveUI '13). ACM, New York, NY,
USA, 150-157. DOI=10.1145/2516540.2516561
http://doi.acm.org/10.1145/2516540.2516561

This data set is opened up to anyone interested in Automotive User Interfaces to 
encourage reproducible results. Please cite our paper if you publish results 
on this data set, and consider making your own data sets open for anyone to 
download in a similar fashion. We would also be very interested to hear back 
from you if you use our data in any way and are happy to answer any questions 
or address any remarks related to it. 


DATA SET CONTENTS
-----------------
These types of data are collected:
Time_Biotrace
Time_Videorating
Time_Light
Time_Accel
Time_GPS
AccelX
AccelY
AccelZ
Lightning
Latitude_GPS
Longitude_GPS
Accuracy_GPS
Altitude_GPS
Speed_GPS
Bearing_GPS
Frame_Videorating
Rating_Videorating
ECG
SCR
Temp
HR
HRV_LF
